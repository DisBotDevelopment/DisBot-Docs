# Placeholder

### How to use

> In the Global Bot System (**Customer and DisBot**) you can use placeholders by typing this: `{<placeholder>}`.  
> But the system has a limit: you cannot use placeholders that tag a member or role in Embed Titles, Authors, Footers, or Timestamps.  
> You can only use image-returning placeholders in image sections of Embeds or Message Content.

---

## Welcome & Leave

| Placeholder            | Description                                            | Where you can use                                    |
| ---------------------- | ------------------------------------------------------ | ---------------------------------------------------- |
| `{member.tag}`         | Replace with the member Tag (`@Jesper`)                | Only in Description and Message                      |
| `{member.name}`        | Replace with the member username (`tnsjesper`)         | Title, Description, Author, Footer, Message          |
| `{member.globalname}`  | Replace with the member Global Name (`Jesper`)         | Title, Description, Author, Footer, Message          |
| `{member.displayname}` | Replace with the member Display Name (`Jesper`)        | Title, Description, Author, Footer, Message          |
| `{member.id}`          | Replace with the member ID (`850470027026759690`)      | Title, Description, Author, Footer, Message          |
| `{guild.name}`         | Replace with the Guild Name (`DisBot™ Development`)    | Title, Description, Author, Footer, Message          |
| `{guild.id}`           | Replace with the Guild ID (`1084507523492626522`)      | Title, Description, Author, Footer, Message          |
| `{guild.membercount}`  | Replace with the Guild Member Count (`28 Members`)     | Title, Description, Author, Footer, Message          |
| `{guild.owner.tag}`    | Replace with the Guild Owner (`@Jesper`)               | Only in Description and Message                      |
| `{guild.owner.id}`     | Replace with the Guild Owner ID (`850470027026759690`) | Title, Description, Author, Footer, Message          |
| `{current.date}`       | Replace with the current Date (`Today at 12:00 pm`)    | In Timestamp                                         |
| `{member.avatar}`      | Replace with the Avatar URL                            | Only in Image, Thumbnail, Author Image, Footer Image |
| `,`                    | New Line                                               | Only in Message                                      |
